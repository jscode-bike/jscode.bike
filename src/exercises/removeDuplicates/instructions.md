# Remove Duplicates

Write a function `removeDuplicates` that accepts a string without spaces and returns the string after removing duplicates. The original order of the characters should remain the same.

```javascript
removeDuplicates("helloworLd"); // should return "helowrd"
removeDuplicates("abbcccddddeeeeeffffffggggggg"); // "abcdefg"
removeDuplicates("theFatCatSatOnTheRat"); // "theFaCSOnR"
removeDuplicates("r3m0v3Dupl1c4te5"); // "r3m0vDupl1c4te5"
```
