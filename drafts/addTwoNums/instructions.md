# Add Two Numbers

Write a function `addTwoNums` that accepts 2 integers and returns the sum of the 2 integers.

```javascript
addTwoNums(3, 5); // should return 8
addTwoNums(100, -5); // 95
addTwoNums(987, 1); // 988
```
